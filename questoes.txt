O que é um repositório no Git e qual a sua importância?
R: Onde ficam salvos os arquivos, histórico e versões do projeto.

Qual a função do Git Bash?
R: É um tipo de terminal que executa comandos do Git e comandos do unix facilitando o uso do git.

O que faz o comando git add e qual a diferença entre git add <arquivo> e git add .?
R: O Git add adiciona mudanças realizadas no projeto. Git add adiciona apenas um arquivo em específico, já o git add . adiciona todas as altrações da pasta atual.

Por que utilizamos o ponto (.) no comando git add .?
R: O ponto representa o diretório (pasta) atual, neste contexto adiciona todas as alterações daquele diretório.

Explique o comando git commit e por que é importante criar uma mensagem descritiva.
R: Registra as mudanças que foram adicionadas, a mensagem é importante para identificar detalhadamente as mudanças realizadas.

Qual a função do comando git push?
R: "Empurrar" as alteraçoes para o main, através do commit realizado anteriormente para o repositório remoto.

Qual a importância de criar commits frequentes durante o desenvolvimento de um projeto?
R: Facilita o controle de alterações e versões, identificação de possíveis erros, e acompanhar a evolução do projeto.

Explique a função das tags HTML e como elas estruturam uma página.
R: As tags dão definição a estrutura e e o conteudo de uma pagina, como: textos, links, imagens e tabelas.

O que são atributos em uma tag HTML e qual a sua função?
R: São informações "extra" que modificam o comportamento ou aparencia da tag.

O que é o Emmet e como ele facilita o desenvolvimento de páginas HTML?
R: É uma ferramenta que auxilia em abreviações no editor de texto e expande em código HTML e CSS, auxilia em economia de tempo de digitação.

O que é o box-sizing no CSS e por que usamos ele no reset inicial?
R: Para definir o tamanho de um elemento calculando a altura/largura e torna o layout mais previsivel. 

Por que é importante utilizar tags semânticas no HTML?
R:As tags semanticas deixam claro o papel de cada parte da pagina, tanto para o navegador quanto para quem está lendo o código

O que são media queries no CSS e qual a sua função?
R: São regras que permitem aplicar estilos deferentes dependendo do tamanho da tela ou dispositivo. Deixa o site Responsivo. 

Qual a função do GitHub Pages?
R: É um serviç do github para hospedar sites diretamente de um repositório. 

O que é o Flexbox no CSS e para que ele é utilizado?
R: é um sistema de layout que facilita a organização de elementos em linha ou coluna, auxilia a distribuir espaços e alinhar itens.

Qual a diferença entre justify-content e align-items em um container Flexbox?
R: O justify-content controla o alinhamento dos itens no eixo principal, se o flex-direction for row esse eixo será horizontal 
o align-items controla o alinhamento no eixo, se o flex-direction for row então esse eixo será vertical.  

Como o flex-grow funciona e qual o seu propósito no layout Flexbox?
R: Define o quanto cada item pode crescer para ocupar o espaço "sobrando" no container. 

Explique a diferença entre flex-direction: column; e flex-direction: row;.
R: Column: organiza os itens em coluna de cima pra baixo. 
row: organiza os itens em linha da esquerda pra a direita.
